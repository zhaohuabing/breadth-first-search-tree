# breadth-first-search-tree
Java implementation of breadth first search for tree data structure

test case
```
                  A
                 /|\
                / | \ 
               B  C  D
                 /\   \
                /  \   \
               E    F   G
                        /\
                       /  \ 
                      H    I
```                      
Result: A B C D E F G H I
